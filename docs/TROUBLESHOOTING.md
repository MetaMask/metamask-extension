# Troubleshooting

This document lists common issues encountered when working on the MetaMask extension
and how to resolve them.

## Extension does not load in browser
- Make sure the build completed successfully
- Reload the extension from the browser extensions page
- Check the background script console for errors

## Changes do not appear
- Ensure you rebuilt the project after making changes
- Hard refresh the extension (disable/enable)
- Clear browser cache if needed

## Network-related issues
- Verify environment variables are set correctly
- Check RPC or Infura configuration
- Confirm the selected network is enabled

If the issue persists, consider opening a GitHub issue with logs and steps to reproduce.
