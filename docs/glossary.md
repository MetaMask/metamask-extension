# Glossary

This glossary explains common terms used in the MetaMask Extension.

## Wallet
A wallet is a tool that allows users to store, send, and receive digital assets.

## Seed Phrase
A seed phrase is a list of words that gives full access to a wallet. It must be kept secret.

## Network
A blockchain network such as Ethereum Mainnet, Sepolia, or other supported networks.

## Gas Fee
A transaction fee paid to validators for processing blockchain transactions.
