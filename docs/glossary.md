# Glossary

A small glossary to help new contributors understand common terms used around the project.

- **dapp**: A decentralized application.
- **EIP**: Ethereum Improvement Proposal.
- **RPC**: Remote Procedure Call endpoint used to connect to a blockchain node.
- **Seed phrase / Secret Recovery Phrase**: A set of words used to restore a wallet.
- **Signer**: A component that can sign messages/transactions.
- **Transaction (tx)**: A signed request to change blockchain state.
- **Token**: A digital asset managed by a smart contract.
- **Vault**: Encrypted storage of sensitive wallet data.
- **Network**: A blockchain environment (mainnet, testnets, custom RPC).
- **Snap**: An extension mechanism for MetaMask (if referenced in the repo/docs).
