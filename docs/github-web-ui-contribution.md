# Contributing via GitHub Web UI (No Terminal)

This short guide helps first-time contributors make a small change using only the GitHub website.

## Steps

1. Fork the repository (button **Fork**).
2. In your fork, create a new branch (**Branch** selector → **Create branch**).
3. Open a file and click **Edit** (pencil icon), or click **Add file** → **Create new file**.
4. Scroll down, write a clear commit message, and click **Commit changes**.
5. Click **Compare & pull request** to open a PR to the upstream repository.

## Tips

- Keep PRs small (one topic per PR).
- Use descriptive branch names like `docs-...` or `fix-...`.
- If you are unsure, open a draft PR and ask for feedback.
