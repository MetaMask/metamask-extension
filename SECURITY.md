# Security Policy

## Supported Versions

We officially support and provide security updates for the following versions of MetaMask:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security-related items.**

If you believe you have found a security vulnerability in MetaMask, please report it via our official Bug Bounty program on HackerOne:

[https://hackerone.com/metamask](https://hackerone.com/metamask)

Our security team will review your report and respond as quickly as possible. We appreciate your help in keeping MetaMask users safe!
