## Navigation Efficiency Criteria

Evaluate how the agent navigates the MetaMask UI:

- Did the agent use direct navigation commands (mm navigate-home, mm navigate-settings) when available instead of clicking through menus?
- Did the agent avoid unnecessary back-and-forth between screens?
- Did the agent use the account list efficiently (not opening/closing it repeatedly)?
- Did the agent take a direct path to the target screen or wander through unrelated pages?
- Penalize: navigating away from the target screen and then coming back to it
