## Onboarding Criteria

When the agent performs wallet onboarding or setup, also evaluate:

- Did the agent complete all onboarding steps without skipping any?
- Did the agent handle the SRP securely (not expose it in tool outputs beyond what's necessary)?
- Did the agent set a password that meets the requirements?
- Did the agent confirm the SRP backup step if prompted?
- Did the agent reach the home screen at the end of onboarding?
